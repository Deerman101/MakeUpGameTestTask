# MakeUpGameTestTask

Тестовое задание, в котором нужно было сделать механику нанесения макияжа на 2D персонажа по готовым ассетам.

## Демонстрация геймплея

https://github.com/user-attachments/assets/1f9d1bd3-02b0-4833-9b46-daf524c5d036

## Что реализовано

— DragAndDrop механика нанесения макияжа выбранным инструментом;

— Инструменты для нанесения макияжа (кисточки: для теней, для румян; помада; спонжик для удаления всего макияжа и крем для лица для удаления прыщей);

— Адаптивая палетка цвета макияжа, можно легко добавлять новые цвета;

— Базовая логика;

— Простые анимации через DoTween.

## Технологии/инструменты

— C# 13 редакции;

— Unity 2022.3.62f3;

— DoTween;

— Unity UI.
